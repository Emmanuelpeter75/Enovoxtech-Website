Analyzing Social Media's Effect on Academic Performance

A deep dive into using Python to uncover the hidden correlations between screen time, platform algorithms, and exam scores.

The Challenge

It is a common assumption that social media negatively impacts academic performance. However, broad assumptions aren't enough when building educational policies or productivity tools. The challenge was to move beyond subjective opinions and use rigorous data science to answer specific questions: At what exact threshold does screen time become toxic to grades? And are all social media hours created equal?

Our Approach

To evaluate this, we needed to map out the relationship between digital habits and academic outcomes. We sourced a comprehensive dataset containing variables like daily study hours, specific social media platform usage, sleep duration, and final exam scores.

"Data science allows us to replace guesswork with precision. We aren't just looking for a decline in grades; we are looking for the exact behavioral threshold where that decline begins."

Data Cleaning & Implementation

Raw data is rarely ready for analysis. The first step was heavy data cleaning using Python's Pandas library to handle missing values and normalize the daily usage metrics. Once the data was structured, we deployed statistical correlation methods to isolate the impact of social media independently of other variables like sleep or baseline intelligence.

python correlation_analysis.py 

# E-NovoxTech Data Analytics Snippet
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load and clean the student dataset
df = pd.read_csv('student_habits.csv')
df.dropna(subset=['social_media_hours', 'exam_score'], inplace=True)

# Calculate Pearson correlation matrix
correlation = df[['social_media_hours', 'study_hours', 'sleep_hours', 'exam_score']].corr()

# Visualize the threshold where screen time impacts scores
sns.regplot(x='social_media_hours', y='exam_score', data=df, scatter_kws={'alpha':0.5})
plt.title('Impact of Daily Social Media on Final Exam Scores')
plt.show()

The Insights Discovered

The visualizations revealed a nuanced reality. The correlation wasn't strictly linear. Usage under 90 minutes a day showed almost zero negative correlation with exam scores. However, a sharp, exponential drop-off in academic performance was identified once a student crossed the 3-hour daily threshold. Furthermore, data showed that late-night usage (after 11 PM) amplified the negative effect by disrupting sleep cycles, proving that when students use social media is just as critical as how long they use it.

These insights demonstrate the power of applied data science: turning raw, noisy behavioral metrics into clear, actionable intelligence.
